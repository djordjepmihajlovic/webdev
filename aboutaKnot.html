The Project

So as we've seen from playing and reading through , it is not always easy to tell two knots apart.

This project builds on previous work, where it was shown that using machine learning methods, some knot measurements allowed for accurate classification of knots!

The Features

In particular, the measurements which lead to highly accurate classification consider how "entangled" every part of the knot is.

Below is an example of high entanglement versus low entanglement for segments of a knot.

Hidden knot invariants

As the machine learning methods are able to accurately classify knots, the project at hand asked whether or not there is hidden information within these "segment entanglement" measurements.

In particular, are there any hidden knot invariants?

This is investigated using machine learning and a host of knot theory mathematics, with surprising results linking our measures of entanglement to knots with points of singularity.
