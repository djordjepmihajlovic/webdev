Welcome to Spot a Knot!

Spot a Knot is a game designed to teach the user about the difficulty in telling knots apart. Knots can be shown to be the same by applying the Reidemeister moves, see for an example of this in action.

The game will show you a knot and you have to guess which knot it is. Spot the correct knots to win!
